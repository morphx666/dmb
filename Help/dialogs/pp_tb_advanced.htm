This is the Toolbars Editor dialog.
It offers all the options needed to create a toolbar within DHTML Menu Builder.
To access this dialog click Tools->Toolbars Editor and click on the Advanced tab.

Click any part of the dialog to get information about its purpose.

Toolbars Editor Options
Adds, removes, and renames Toolbars.

  • Add (): Adds a new toolbar.
  • Remove (): Removes the selected toolbar.
  • Rename (): Renames the selected toolbar.

Follow Scrolling
Configures the toolbar to follow the browser's scrollbar position.

Checking the left-right arrows will cause the toolbar to follow the horizontal scrollbar.
Checking the top-bottom arrows will cause the toolbar to follow the vertical scrollbar.

The Smart Scrolling feature controls the behavior of the scrolling feature. When enabled, the toolbar will only scroll when needed, that is, when it's about to go beyond the visible area. If disabled, the toolbar will always scroll.

Compatibility notes
This setting has no effect with Opera 5.12

Toolbar Size
Controls the default size of the toolbar.

By default, DHTML Menu Builder will calculate the optimal width and height of the toolbar based on the toolbar contents. 
Use these settings to force the width and/or the height of the toolbar to be different.

For the width you have three options:

  • Auto: Optimal width is calculated by DHTML Menu Builder.
  • Match Group's Width: Toolbar item to have the same width as the group it is set to display. Note that this mode is only available when the toolbar alignment is set to Horizontal.
  • Manual: Toolbar width is typed in. Use the Calculate button to let DHTML Menu Builder calculate the optimal width. Then adjust it manually.

For the height you have two options:

  • Auto: Optimal height is calculated by DHTML Menu Builder.
  • Manual: Toolbar width is typed in. Use the Calculate button to let DHTML Menu Builder calculate the optimal width. Then adjust it manually.

This is the Special Effects dialog.
This dialog applies only to groups, but its settings can affect the child commands of the selected group.
To access this dialog select a group, click Menu->Margins, click on the Group Size tab and click on the Scrolling button..

Click any part of the dialog to get information about its purpose.

Enable Scrolling
Checking this option will enable the scrolling support.

Compatibility notes
The scrolling support will only work on DOM-compliant browsers.
No support is currently available for Navigator 4.
Style Settings
Here you can configure the look of the two arrows used to scroll the menus.

The controls under the Normal pane control how the arrows will look when the mouse is not over them. The Over pane controls how the arrows will look when the mouse is moved above them.

The supported styles are:

  • Background color
  • Arrow images
Margin
This control will let you set a margin (blank space) for the arrow buttons.
Scroll when the mouse is over the scroll buttons
If this option is enabled the menus will start scrolling as soon as the mouse passes and stays over the scroll buttons.
When this option is disabled, the user will have to click the buttons in order to scroll the menus.

This is the New Window Parameters dialog.
This version of the New Window Parameters dialog applies to both Groups and Commands.
To access this dialog select a menu item and click on the Parameters icon ().

Click any part of the dialog to get information about its purpose.

Window Name
DHTML Menu Builder assigns a name to each window. You can change it if you wish, but it is usually not necessary.

Position and Size
Indicate where you want the upper left corner of the new window to be placed on the screen. You can also indicate the width and height of the window.

These settings are measured in pixels by default. You can specify relative values by following the number with a % sign.
For example, specifying a width of 50% will make the new window half the size of the available screen width.
You may also leave any of these settings empty (blank) so the new window is created with the client's default values.

Select the "Center on screen" option if you want to have the dialog appear centered on the screen regardless of the screen size (resolution).

Note that no verification will be done, so if the position places the window outside the visible area of the user's screen, the window will not be seen.

Options

There are several options that control the appearance of the new window. Checking the box next to each option enables it:

  • Display directory buttons: In Internet Explorer, displays the Links bar; in Mozilla-based browsers (Netscape Navigator 6, etc.) is shows the Personal Navigation bar.
  • Channel Mode: Display the window in theater mode and show the channel band.
  • Display Toolbar: Show/hide the browser toolbar, making buttons such as Back, Forward, and Stop available.
  • Display in Full Screen: Display the browser in a full-screen window with no visible controls.
  • Display the input field: Show/hide the input field for entering URLs directly into the browser.
  • Display the menu bar: Show/hide the menu bar.
  • Allow resizing: Show/hide display resize handles at the corners of the window.
  • Display scrollbars: Show/hide horizontal and vertical scroll bars (if necessary)
  • Display status bar: Show/hide the status bar at the bottom of the window.
  • Display title bar: Show/hide the title bar for the window.

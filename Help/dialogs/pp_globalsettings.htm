This is the Special Effects page of the Global Settings section of the Project Properties dialog.
Settings that will affect the way your menus behave are configured here.
To access this dialog click File->Project Properties and click on the Global Settings tab.

Click any part of the dialog to get information about its purpose.

Unfolding Effect
Menus built with DHTML Menu Builder can be displayed with various effects. Use the drop-down list to select the effect to use.

Effects slow down the menus somewhat, so if you want your menus to appear as fast as possible, select (none) and no effect will be used. Use the Effect Speed slider to vary the speed of the selected effect.

Menu Items Blinking Effect
Here you can specify the amount of blinking that you want to occur when you click a menu item.

The Amount parameter will control how long with the menu item will remain blinking and the Speed parameter will control the speed of the effect.

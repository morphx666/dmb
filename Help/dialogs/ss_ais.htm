This is the Advanced Item Selection dialog.
To access this dialog select any menu item and then open any of the style dialogs; click on the Scope Selector control and click the Advanced option.

Click any part of the dialog to get information about its purpose.

Filter
In this section is where you define the criteria to filter the menus items.

The available filters are described below:

  • Enabled: When this option is ticked, only enabled items will be shown. If the option is un-ticked, then only disabled items will be shown. If the option is grayed, then both enabled and disabled items will be shown.
  • Display a Submenu: When this option is ticked only commands that display a submenu (or subgroup) will be shown.
  • Belonging to the Toolbar: Here you can filter menu items that belong to a particular toolbar. If the option "any" is selected, then all menu items will be shown.
  • Caption Contains: In this space you can type several letters and only those menu items that contain these letters in their caption will be shown.
  • Uses the Font: You can use this filter to include those items with a specific font settings.
  • Uses the Image: Choosing one or more of the available images in the list will limit the amount of shown items to those that contain the selected images.

Selection buttons
These buttons provide an easy and fast way to make modifications to the list of selected items.

  • All: Clicking this button will select all the shown menu items.
  • None: Clicking this button will un-select all the shown menu items.
  • Inv: Clicking this button will reverse the selected state of all the show menu items.

Menu Items List
In this list will be shown those menu items that match the criteria of your filter.

Then, you can further filter down the menu items that you want to select by only checking those that you need.

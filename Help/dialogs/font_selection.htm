 Dialogs » Production » Font Selection

This is the Font Selection dialog.
Select font, font style, and font size for the selected item.
To access this dialog select a group or a command and click Menu->Font and then click on one of the Change buttons.

Click any part of the dialog to get information about its purpose.

Name

Style
Check the style to apply to the selected font.

Size
Select the size, in pixels, to apply to the selected font.

In case the font size you want to use is not listed, you can also type the font size directly .

Compatibility notes
Using a font size not available on the list may cause the menus to render improperly under Netscape Navigator 4 for the Macintosh and Linux operating sytems.

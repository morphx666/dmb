Plastic Green Button

Implementation instructions

  • Set the group margins (Menu->Margins) to 0px
  • Under the Menu->Special Effects
    • Border size to 0px
    • Horizontal Margin to 17px
    • Vertical Margin to 16px
  • Under the Menu->Special Effects->Group Size
    • Width set to Manual
    • Manual value set to 114
  • Commands' background color must be set to transparent
  • Commands' fore color looks best with brighter colors
  • Commands' font should be set to Tahoma, 11, Bold
    You will need to adjust the Vertical Margin from the Special Effects dialog for other font settings

This is the Item Highlight Wizard dialog.
This dialog is used to implement what is known as automatic item highlighting which is very useful if you want to reflect, in the menus, the section of your web site where you users are currently located.
In the screenshot shown below, the menu item "Downloads" will be automatically highlighted when the "download.htm" document is loaded, allowing the visitor of the web site to see where he/she currently is.

Click any part of the dialog to get information about its purpose.

Files
Displays the list of files on your web site.

Only files that can receive the code required to highlight a menu item will be displayed. A valid file must meet the following conditions:

  • The file must be a valid HTML document.
  • The file must already contain the menus.

Files displayed using a blue color indicate that these files already contain the code to highlight an item.

Menus
Displays the menus on the project.

Note that some items, such as separators, will not be shown on this list. This is just because these items cannot be highlighted.

Information Pane
Displays information that will guide you through the process of installing the code that highlights menu items.

Install / Remove buttons
These are the control buttons used to either install or remove the highlighting code from the documents on your web site.

To install the code on a document, first select the document from the Files list and select a menu items from the Menus pane.
Then, click the Install button.
To install the code on all the documents pointed by the links on your menus click the small arrow next to the Install button and select Install to all links in Menus.

To remove the code from a document, simply select the document from the Files list and click the Remove button.
To remove the code from all the documents in the web site click the small arrow next to the Remove button and select Remove All.

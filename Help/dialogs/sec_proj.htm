This is Secondary Projects dialog.
This dialog will let you specify the secondary projects for the currently loaded (master) project.
To access this dialog click Tools->Secondary Projects

Click any part of the dialog to get information about its purpose.

Projects
Displays the selected secondary projects.

Notice that each secondary project can be checked and unchecked.
Checked projects will be included in the following the tasks when selected:

  • Compile: Secondary projects that have been checked will be compiled. Those that have not been checked will not be compiled.
  • Install: All the secondary projects that have been checked will be installed using the Install Menus tool.

Different types of projects will be represented differently in the Secondary Projects list:

  • Projects from older versions of DHTML Menu Builder will be dimmed and you will not be able to select them.
    To upgrade an old project simply open it in DHTML Menu Builder and click File->Save
  • Projects that do not exist or projects that use frames support will be displayed in red and you will not be able to select them.

Compile and Install
These buttons are shortcuts to the most common functions used with secondary projects.

When clicking the Compile button, the master project and all the checked secondary projects will be compiled.
When clicking the Install button, the master project and all the checked secondary projects will be installed using the Install Menus tool.

Add, Delete and Select All buttons
These buttons will let you add and delete projects from the Secondary Projects list.

Use the Select All button to check (select) all the projects at once. Any task that is performed after selecting all the projects will be applied to all the secondary projects.

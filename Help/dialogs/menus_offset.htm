 Dialogs » Project Properties » Advanced » Menus Offset

This is the Menus Offset dialog. It is found in the Advanced section of the Project Properties dialog.
In this dialog you can modify the position of the menus by applying an offset to their default values.
To access this dialog click File->Project Properties, click on the Advanced tab and then click on the Menus Offset button.

Click any part of the dialog to get information about its purpose.

Root level Groups
The settings in this section control the offset of the root level groups relative to their parent hotspots.

To offset the menus to the right and/or to the bottom of the page use positive numbers.
To offset the menus to the left and/or to the top of the page use negative numbers.

Offsetting the default position of the root level menus is specially useful when the menus are implemented in a framed page and some frame is pushing the menus to the right or towards the bottom of the page.
For example, if you have a frame at the top of the page that is pushing the menus down, then simply add the height of the top frame, in negative numbers, to the Pixels Vertically text box.

Cascading Groups (Submenus)
The settings in this section control the offset of the groups relative to their parent commands.

To offset the menus to the right and/or to the bottom of the page use positive numbers.
To offset the menus to the left and/or to the top of the page use negative numbers.

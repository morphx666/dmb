Search Engine Optimizations

By default, the menus generated by DHTML Menu Builder are not visible to search engines. This means that if the menus are the only means of navigation on your web site, search engines will not be able to follow the links on your menus and your web site will not be properly indexed.
To resolve this problem you need to enable the "Search Engine Optimizations" option under the File->Project Properties->Global Settings->Extended Features dialog.

When this option is enabled, DHTML Menu Builder will generate a plain, HTML-based, version of your menus which search engines will be able to use to properly index your web site. This alternate navigation will remain completely hidden from your visitors and will only become active when a search engine visits your web site (or a user visits your web site with javascript disabled).

Note that for this option to work you must re-install the menus, as described in this document.

If you want to further improve the chances of search engines indexing your web site you should also consider creating a sitemap and linking to it from the front page on your web site.
DHTML Menu Builder can create sitemaps from your project using the File->Export As HTML option.

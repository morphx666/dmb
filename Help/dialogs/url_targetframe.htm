This is the URL Target Frame dialog.
This dialog will let you specify the frame in which you will load the link of the selected item.
To access this dialog select a menu item and click on the target frame icon ().

Click any part of the dialog to get information about its purpose.

Available Frames
Displays the four most common frames:

  • _self: The default frame for all the menu items. It points to the same page on which the menus are being displayed.
  • _top: Loads the link on top of any frameset currently loaded into the browser.
  • _blank: Displays the link in a new window. Since this is technically a new frame, it is mentioned here. However, DHTML Menu Builder also has a separate command that allows you to display links in new windows. It is much more flexible and provides more features than the "_blank" target frame. More information.
  • _parent: Displays the selected link in the immediate parent frame from where the menus are being viewed.

This list will also include the frames, if any, detected in the Frames Document.

Target Frame
Displays the name of the selected frame.

To select a frame, either choose one from the Available Frames list or type it in this text box.

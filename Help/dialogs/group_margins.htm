This is the Margins dialog.
This dialog applies only to groups, but its settings actually affect the contents of the selected group.
To access this dialog select a group and click Menu->Margins

Group Contents' Margins
Controls the inner margins of a group's contents:

  • Horizontal margin is the distance from the inner part of the group's border to the outer part of the command's left and right borders.
  • Vertical margin is
    • the distance from the inner part of the group's top border to the outer part of the topmost command
    • the distance from the outer part of the bottommost command's border to the inner part of the group's inner part of the bottom border.

The Scope Selector
This advanced control will let you select which item or items should receive the styles on the dialog.

By default, only the selected item will receive the style changes, however you can choose from a variety of options to apply the same style changes to a set of menu items all at once.

This is the Images Browser dialog.
This dialog offers a convenient way to access all the images used in your project.
To access this dialog select a group or a command and click Menu->Image and then click on one of the Change buttons.

The images are actually stored with your project file so others can use the images without actually having them on their local machine.

Click any part of the dialog to get information about its purpose.

Images Browser
Displays a list of all the images used on your project.

To select an image from the list, select the image and click OK.

The image currently in use, if any, appears with a sunken border.
The selected image appears with a thin black border.
If the filename for an image is shown in red, that indicates that the image is no longer in the same location as it was when it was added to the project. Use the Update Location button to find the file and update its location.

From File button
Click to select an image that does not appear in the list.

If you decide to use a new image, it will be added to the list so you can access it later without having to browse your computer to find it.

Update Location
Click to update the location of the selected image.

Because all the images are stored in the project file, deleting or moving the source image that you initially selected does not make it unavailable to your project. However, if too many files are missing, DHTML Menu Builder can slow down  considerably.

To avoid this, use the Update Location button to update the location of files whose name are displayed in red. When DHTML Menu Builder uses the file from your hard disk instead of the project file, the program works faster.

Save button
Click this button to save the selected image.
The image will be extracted from the project and saved into the specified folder.

Extract Icon
Click this button to access the Extract Icon tool.
This tool can be used to extract icons from programs (EXE), dynamic link libraries (DLL), icon libraries (ICL) and any other file type that may contain icons.

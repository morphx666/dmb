This is the URL Bookmark dialog.
This dialog will let you define bookmarks so you can link to a named section on the linked document.
To access this dialog select a menu item and click on the bookmark icon ().

Click any part of the dialog to get information about its purpose.

Available Bookmarks
Displays the named areas, if any, in the linked document.
Select the named area to bookmark it.

Target Bookmark
Displays the name of the selected bookmark

To select a bookmark either choose one from the Available Bookmarks list or type it in this text box.
To remove the bookmark, leave this text box empty.

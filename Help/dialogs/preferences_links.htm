Verify Links When
These settings will let you automate the links verification process.
You can specify to either check for broken links when opening a project, when saving it or when compiling it.

You can also specify if you want to the check to include external links, that is, links that point to a web server outside from the server where the menus are installed on.
Note that enabling this option can considerably slow down the Opening/Saving/Compiling process if your project contains many external links.

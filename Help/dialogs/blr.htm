This is DHTML Menu Builder's Broken Links Reports dialog.
This dialog can be used to verify the links on all menu items, both internal and external and make changes to fix any broken links.
To access this dialog click Tools->Reports->Broken Links Report

Click any part of the dialog to get information about its purpose.

Links Report
This list displays the links from all the active menu items in the project.

The list is divided in two columns:

  • The first column shows the caption of the menu item and an icon representing its status:
    • The link has not been tested
    • The link has been tested and its valid
    • The link has been tested and its broken
  • The second column shows the menu item's link

Edit button
Click this button to edit the link of the selected item.

Test Selected and Test All buttons
These buttons are used to test the links on the links report list.

  • Click the Test Selected button to test the selected item.
    Use the CTRL and/or SHIFT keys to select multiple items
  • Click the Test All button to test all the items in the list report

Previous and Next buttons
Click these buttons to quickly jump to the previous or next broken link in the links report list.

Filter buttons
These buttons are used to filter the links displayed on the links report list:

  • When pressed, the list will include the "untested" items
  • When pressed, the list will include the "tested and valid" items
  • When pressed, the list will include the "tested and broken" items

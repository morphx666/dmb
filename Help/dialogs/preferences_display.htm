Menu Items Style
This section defines how the standard menu items will be displayed on the "Map View".

Use the Color button to specify the color and use the Bold button to specify whether if the menu item should rendered using a bold style or not.

Special Items Styles
This section defines how the special menu items will be displayed on the "Map View".
Special menu items include:

  • Items that are disabled
  • Items that contain a broken link
    NOTE: For a more reliable and comprehensive links verification use the Broken Links Report tool.
  • Items excluded from compilation

Use the Color button to specify the color and use the Bold button to specify whether if the menu item should rendered using a bold style or not.

Defaults button
Click this button to reset the styles of all menu items to the program's default values.

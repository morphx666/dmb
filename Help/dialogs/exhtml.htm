 Dialogs » Application » Export As HTML

This is DHTML Menu Builder's Export As HTML dialog.
This is where you configure the settings that control the generated HTML code.
To access this dialog click File->Export As HTML

Click any part of the dialog to get information about its purpose.

Heading
Defines the title and description that will appear in the heading of the HTML document.
This information will also be used on the HTML head section of the document.

Style
Defines the basic style of the generated HTML output:

  • None: The text, color and other styles that apply to the generated code will be determined by the default settings of the browser.
  • Use Project Font and Color Styles: Applies the styles from your groups and commands to the HTML elements so they look similar to the menus.
  • Use External CSS File: Applies the styles from your own style sheet. DHTML Menu Builder will link to it and let you determine which classes should be applied to the items on the HTML document. Note that the .css document must be part of your web site, i.e., located inside the root web.

Paths
Allows you to specify the locations where DHTML Menu Builder should save the files it generates.

  • HTML File Name: Specifies the location where the generated HTML document containing your menus will be stored.
  • Images Path: Allows you to specify one of two locations for the images used in your menus:
    • the same location as defined on your default configuration
    • an alternate folder

Preview
Click this button to generate preview and see how the HTML version of the menus will look and feel like.

Generate
Click this button to generate the HTML document containing the HTML version of your menus.

This is the Remove Loader Code dialog.
It presents a convenient way to remove the loader code from your files.
To access this dialog click Tools->Remove Loader Code

NOTE: This dialog is only available when the "Use Advanced Menus Installation Tools" option is selected.

Click any part of the dialog to get information about its purpose.

Files containing the Loader Code
Lists all the files that contain the loader code.

By default, DHTML Menu Builder will examine the site and recommend a set of files that include this code. You can add more files by using the Add Files and Add Folder buttons.

Start
Starts the removal process.

Add Files, Add Folder and Remove Buttons
Adds more files to the list.

Use Add Files to add one or more files from the same directory.
Use Add Folder to add all the files inside a folder and all the subfolders underneath.
Use Remove to remove a file from the list. You can also select a folder and click Remove to delete all the files and folders underneath.

Note that the dialog will automatically filter out, any files that do not contain the loader code.

This is the Advanced Offsets Definitions dialog.
In this dialog is where you create custom conditions that determine when to apply an offset, to which elements will be applied and how much.

Click any part of the dialog to get information about its purpose.

Description
This is a place where you can type a small description explaining what this definition does.
Note that this is a required parameter.

Operating System
Use this setting to limit under which operating system the condition should be applied.

The available options are:

  • Any: This indicates that the condition will run under any operating system
  • Windows: The condition will run under Windows only (this includes all versions of Windows).
  • Linux: The condition will run under operating systems other than Windows and Mac.
  • Mac: The condition will run on Mac machines running any version of System OS, MacOS and MacOS X.

Browser
Use these settings to control what browsers will the condition be applied to.

Browser Version
Here you can further limit the number of browsers supported by the condition by filtering only those browsers that match a particular version number.

Offsets
In this section you define which elements should be offset and by how much.

The offsets specified here will only be applied when the condition you defined is true.

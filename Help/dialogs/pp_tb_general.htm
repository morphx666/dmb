This is the Toolbars Editor dialog.
It offers all the options needed to create a toolbar within DHTML Menu Builder.
To access this dialog click Tools->Toolbars Editor and click on the General tab.

Click any part of the dialog to get information about its purpose.

Toolbars Editor Options
Adds, removes, and renames Toolbars.

  • Add (): Adds a new toolbar.
  • Remove (): Removes the selected toolbar.
  • Rename (): Renames the selected toolbar.

Groups List
Displays the groups assigned to the selected toolbar.

Control Buttons
Adds groups to and removes groups from the selected toolbar. Also sets the order of the groups in the toolbar.

  • Add (): Displays a list of all the available groups. Click a group to add it to the selected toolbar. Groups that are already assigned to the selected toolbar appear checked.
  • Remove (): Deletes the selected group(s).
  • Move Up (): Moves the selected group up one position.
  • Move Down (): Moves the selected group down one position.

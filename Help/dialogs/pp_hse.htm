This is the Target Document configuration section.
If you have created your own Toolbar or navigation bar, the HotSpots Editor needs this information so you can attach the menus.
To access this dialog click File->Project Properties, click on the Configurations tab and then click on the Target Document button.

Click any part of the dialog to get information about its purpose.

Target Document
Full path to the HTML document that contains the toolbar that will trigger the menus.

This document can have any file extension such as .htm, .asp. It can even be template (such as .dwt) or an include file of (such as .inc). The only requirement is that this document must contain the hotspots that will receive the trigger code to display the menus. For more information on how to prepare your hotspots so the HotSpots Editor can attach the code, see the section Preparing your Web Site.

NOTE: DHTML Menu Builder will modify this document and add the trigger code to your hotspots. It is a good idea to always keep a backup of this file.

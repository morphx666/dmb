Create Collapsible Tree
Tick this option to enable the creation of collapsible tree-like menus.
 
Compatibility notes
The tree-like navigation structure created with this option will only work on DOM-compliant browsers such as Internet Explorer 5.0 and above, Opera 7 and Navigator 6 and above including all Mozilla-based browsers.

Images
In this section you define the images that will be used in the tree-like menu.

These images are often used to indicate the state (expanded, collapsed or normal) of a tree-menu item.

In this section you can also define the indentation of sub-tree menu items.

Options
Here you can specify several options that will modify the behavior of the tree-like menus.

  • Expandable Items can have links: If ticked, menu items that can be expanded will also contain links. So, to expand a menu item you will have to click on its image, because clicking on its caption will trigger link. If this option is un-ticked, clicking on the image or the caption will expand the menu item and the link information will be ignored.
  • Single Select: When this option is ticked, only one branch of the tree will be expanded at the same time. Expanding another item from the same level will collapse all the menu items from that same level.
    This is very useful for large projects.
  • Include Expand/Collapse All options: When enabled, an "Expand All | Collapse All" will be added to the document so users can expand or collapse the whole tree at once.

Defaults
Clicking this button will reset all the options on this tab to their default values. The images will also be re-set to their default [+], [-] and [•] icons.

Preview
Click this button to generate preview and see how the HTML version of the menus will look and feel like.

Generate
Click this button to generate the HTML document containing the HTML version of your menus.

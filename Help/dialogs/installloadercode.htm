This is the Install Loader Code dialog.
This dialog offers a convenient way to install, into your pages, the code used to load the compiled files.
To access this dialog click Tools->Install Menus->Install Loader Code and from the frames version click Tools->Install Menus->Install Frames Loader Code

There're two versions of this dialog. One is used to install the standard loader code and the other is used to install the loader code for web sites using frames.
NOTE: This dialog is only available when the "Use Advanced Menus Installation Tools" option is selected.

Click any part of the dialog to get information about its purpose.

Files to install the loader code
This lists all the files that will receive the loader code.

By default, DHTML Menu Builder will recommend a set of files that must include this code. You add more files using the Add Files and Add Folder buttons.

Install Button
Click this button to start the installation process.

Add Files, Add Folder Add Links and Remove Files Buttons
Use these buttons to add more files to the list.

  • Use the Add Files button to add one or more files from the same directory.
  • Use the Add Folder button to add all the files inside a folder and all the subfolder underneath.
  • The Add Links button is only available on the Install Loader Code dialog. When you click on the button all the documents on your web list that are linked from the menu items on your project will be added to the dialog. This is particularly useful when using a configuration of type CD-ROM and you need to install the loader code on all the documents of your web site that will display the menus.
  • Use the Remove Files button to remove a file from the list. You can also select a folder and click the Remove Files button to delete all the files and folders underneath.

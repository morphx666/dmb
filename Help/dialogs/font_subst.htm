 Dialogs » Project Properties » Advanced » Fonts Substitution

This is the Fonts Substitution dialog found in the Advanced section of the Project Properties dialog.
Use this dialog to create families of font substitutions for the main fonts used by the items in the loaded project.
To access this dialog click File->Project Properties, click on the Advanced tab and then click on the Fonts Substitution button.

Click any part of the dialog to get information about its purpose.

Substitutions
Lists the fonts you want the browser to try to use when the main font is not found.

Add button
Click Add to add possible font substitutions for the selected main font.

You can also edit this list manually. The font names must be separated by commas. Watch the spelling; as always, it is better to select the font from a list than type it manually.
To delete a substitution, select the font and press the DEL (delete) key on your keyboard.

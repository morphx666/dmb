 Dialogs » Production » Install Menus (Frames Version)

This is the Install Menus dialog.
This dialog offers a convenient way to install the menus into the pages of your web site.
To access this dialog click Tools->Install Menus

When not using frames support this dialog will look slightly different.
NOTE: This dialog is only available when the "Use Advanced Menus Installation Tools" option is not selected.

Click any part of the dialog to get information about its purpose.

Folders on web site
This pane displays all the folders on the web site.

You can drag any folder in the left pane to any of the right panes to install the menus in the files located in the dragged folders.
You can also right click on any folder to display a list of actions appropriate for the files in the folders.

Files On ...
This pane displays the list of files on the currently selected folder.

Each file status is represented in different colors:

  • Light Gray: cannot receive the menus.
  • Black: can receive, but does not contain any menus.
  • Blue: already contains the standard loader code used to load the menus.
  • Green: already contains the frames loader code use to load the menus.

You can drag files to any of the right panes to install the menus in the selected list of files.
You can also right click on any number of files to display a list of actions appropriate for the selected files.
 

Files that will receive the Toolbar
Usually there should only be one file that receives the toolbar. This file, when displayed on the browser will load the toolbar and display it.

If you are using hotspots, instead of toolbars, then this is the file containing the hotspots.

To add files to this list, either drag them from the Files pane, the Folders pane, or click on the Auto button.
To remove files from this list, either right click on the file and choose "Remove from List" or press the Delete key.

Auto
Clicking this button causes DHTML Menu Builder to start a scan of all your files in your web.  DHTML Menu Builder automatically decides which files must receive the loader code either because they need it, or because it is out-of-synch with the project.

After the Auto Detection process finishes, examine the results and make sure that the list of files queued to receive the menus is correct.

Install
Click Install when you are ready to install the menus in the list of selected files.

Files that will receive the Menus
This is the list of files that will receive the menus.
These files will contain all the information needed to display the menus when triggered from the toolbar.

To add files to this list either drag them from the Files pane, the Folders pane or click on the Auto button.
To remove files from this list, either right click on the file and choose "Remove from List" or press the Delete key.

Filters
These filters are used to adjust the files displayed on the Install Menus dialog:

  • Hide Unsupported Documents: When enabled, the Install Menus dialog will only display known type of documents that support displaying the menus. When un-selected, all the documents in the web site will be displayed.
  • Include FrontPage SubWebs: By default, the Install Menus dialog will detect when a folder belongs to a FrontPage subweb and will ignore its contents. Enable this option to turn off this filter and display folders that contain FrontPage subwebs.

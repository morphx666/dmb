Presets
A preset is a specialized project configured to server as either a starting point to create a new project or to use it as a template to apply its styles to another project.

DHTML Menu Builder uses the Presets Manager dialog to let you work with any Presets that you may have installed.
The Presets Manager offers options to let you organize your presets, create new projects from any available preset and it also let's you select which styles should be imported from a preset into an existing project.

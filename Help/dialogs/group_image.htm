This is the Image dialog.
This version of the Image dialog applies only to Groups.
To access this dialog select a group and click Menu->Image

Click any part of the dialog to get information about its purpose.

Back Image
Controls the image that appears as a background for the group.

Note: there is no way to control the size of the image. It will be tiled to cover the whole space occupied by the group.

Compatibility notes
Netscape Navigator 4 may have problems displaying non-JPG images in the background.

Live Preview
Allows you to see how the selected item will appear in your browser.

The Live Preview is updated at the same time you make changes to your items. That way, you always know how your menus will appear without having to go through the process of compiling your project or even previewing.

When a group is selected, the Live Preview shows how the background of the menu will appear. If the toolbar option is selected, the preview will show how the actual toolbar hotspot will appear.

When a command is selected, the Live Preview shows how the selected command will appear against its parent group.

Borders: Images
Controls the color of the borders that surround the group.

To change the image of a border, click the border and then click the Change button.

The state of each border is represented by a color:

  No image assigned
 Selected, but no image assigned
 Image is assigned
 Selected, and image assigned

The use of images in the four corners is optional. When a corner is left without an image, its space will be covered by the image from one of the closest main border.

Compatibility notes
Netscape Navigator 4 may have problems displaying non-JPG images in the four main borders.
When using images in the borders of a group, Netscape Navigator 4 will not display the border color, although the size of the border will be taken into consideration to render the menu.

In the Opera browser, submenus will appear incorrectly aligned when using images in the borders.

Under some circumstances, the Live Preview will not display the borders properly.
When using images in the borders, the Live Preview can considerably slow down DHTML Menu Builder's performance.

The Scope Selector
This advanced control will let you select which item or items should receive the styles on the dialog.

By default, only the selected item will receive the style changes, however you can choose from a variety of options to apply the same style changes to a set of menu items all at once.

This is the HotSpots Editor dialog.
In this dialog you attach each of your groups to their respective hotspots.
To access this dialog click Tools->HotSpots Editor

Click any part of the dialog to get information about its purpose.

Available HotSpots
This is the list of all the hotspots detected in the loaded document.

Each item will display an icon representing the type of hotspot:

More information on how to prepare your documents so DHTML Menu Builder can detect the hotspots.

Attached Group
Shows the group currently attached to the selected hotspot.

You can change the group by selecting another one from the list.

To detach a group from the selected hotspots, select (none) from the list.

Alignment
Determines the alignment of the menus relative to the selected group (hotspot)

Image Name
If the selected hotspot is an image, it is displayed here.

If the selected hotspot is an image and it has no name, type a unique name for it. When your image-based hotspots have a name, DHTML Menu Builder can implement auto relative positioning. With this, the menus will always appear aligned to your hotspots regardless of the position of the hotspots. Instead of using fixed values to position the menus in your pages, DHTML Menu Builder will determine the position of the hotspot at the moment it is triggered. The menus will appear perfectly aligned to the image that triggered it.

Position
Sets the absolute position of the selected group when the selected hotspot is triggered.

This setting is only available when the selected hotspot is text-based.
The numbers in the text boxes will specify the left and top coordinates of the menu.

Image Preview
If the selected hotspot is an image, a preview will be shown in this space.

Active Text
Converts a text-based hotspot into an active element so DHTML Menu Builder can implement auto-relative positioning. This button is enabled only when the hotspot is text. The result is that DHTML Menu Builder can determine the exact position of the hotspot at when it is triggered and automatically align the menu to the hotspot. This will occur even if the position of the hotspot changes when the browser is resized.
